## Инструкция по запуску

Быстрый старт через Docker Compose:
* ```bash 
  cp .env.example .env
  ``` 
  создает файл `.env` на основе шаблона `.env.example` и заполняет его значениями по умолчанию. \
  \
  Если нужно изменить какие-то переменные окружения, отредактируйте файл `.env` после его создания.

* ```bash 
  make run
  ``` 
  поднимает сервис на `:8080` внутри контейнера, но за пределами контейнера, надо обращаться к порту `:8082` (например для запросов через curl/Postman)
  
* ```bash 
  make logs
  ```
  посмотреть логи сервиса
* ```bash 
  make down
  ```
  остановить и удалить контейнеры
Сервис слушает порт 8083, если такой порт уже занят, порт можно изменить в .env

## Требования
Docker/Docker Compose, Go 1.26 (для запуска без контейнеров).
